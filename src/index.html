🔧 手动抓包下载

        
适用于重度定制化的小鹅通店铺（链接常带有 decorate/homepage 等字样），需要手动从浏览器抓包获取下载信息。


        

          

            ❶
            

              打开视频播放页
              
在电脑浏览器中打开你要下载的课程视频页面。

            

          

          

            ❷
            

              获取 userId
              
按 F12 打开开发者工具 → Network 面板 → 搜索 pushData → 在 Payload 中找到 userId。

            

          

          

            ❸
            

              获取 m3u8 链接
              
在 Network 面板搜索 m3u8（可先点击播放按钮触发请求），复制请求的 URL。

            

          

          

            ❹
            

              填入下方表格
              
将 userId 和 m3u8 URL 填入下方，点击 Start 即可下载。

            

          

        


        

          💡
          复制 URL 时可以直接带上 "请求网址" 前缀，程序会自动清理。每个视频需要单独抓取 m3u8 地址。
