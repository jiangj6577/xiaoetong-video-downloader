手动抓包指南

            
适用于重度定制化的小鹅通店铺（链接常带有 decorate/homepage 等字样）。

          

          

            

              
1

              

                打开视频播放页
                在电脑浏览器中打开你要下载的课程视频页面。
              

            

            

              
2

              

                获取 userId
                按 F12 打开开发者工具 → Network 面板 → 搜索 pushData → 在 Payload 中找到 userId。
              

            

            

              
3

              

                获取 m3u8 链接
                在 Network 面板搜索 m3u8（可先点击播放按钮触发请求），复制请求的 URL。
              

            

            

              
4

              

                填入并导出列表
                将获取到的 m3u8 URL 填入下方表格，点击“导出”保存当前下载清单。
