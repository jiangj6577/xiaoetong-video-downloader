# 🦆 小鹅通 M3U8 视频全自动解析导出工具 (Xiaoetong M3U8 Parser & Exporter)



> 一款基于 **Electron** 打造的全自动小鹅通（Xiaoetong）视频、专栏 M3U8 地址解析与导出工具。
> **⚠️ 注意：本项目仅作为“嗅探解析器”提取真实的视频播放地址（M3U8），不内置视频下载和解密合并功能。** 导出的下载地址列表，可供专业的第三方下载工具（如 `N_m3u8DL-RE` 等）进行高效的批量下载与解密。
>
> *An automated Electron tool that scrapes and extracts real M3U8 video URLs from Xiaoetong courses, exporting them as a text file for external downloaders.*

---

## ✨ 核心功能

- 🤖 **全自动目录解析**：支持模拟滚动加载、自动下钻“专栏”和“大专栏”的多层级目录，自动获取整个课程的所有视频节点。
- 🔍 **精准提取 M3U8**：通过接口数据拦截、Vue 实例深度探测、网络性能 API 嗅探等多种混合方案，精准定位加密和非加密的真实 M3U8 地址。
- 💾 **一键导出下载列表**：将解析完毕的所有视频标题和 M3U8 链接，快速导出为系统下载目录下的 `m3u8-list.txt` 文件，方便无缝对接第三方下载器。
- 📦 **开箱即用的 UI**：提供清晰直观的桌面客户端操作界面，无需配置复杂的抓包环境和抓包代理。

## 🛠️ 环境要求

- **[Node.js](https://nodejs.org/)**: 建议使用 `v18` 或以上版本。

## 🚀 安装与启动

克隆或下载本项目源码后，在项目根目录下执行：

**最简单的启动方式**（首次运行会自动拉取 Electron 环境）：
```bash
npm install
npm start
```

*如果你希望以后启动更快，可以先在本地安装 Electron 依赖：*
```bash
npm install -D electron
npm start
```

### 🟢 场景一：标准课程/专栏的自动化批量解析导出

对于绝大多数**未经过重度定制**的标准小鹅通课程页面，本工具支持一键化提取整个列表：

1. **输入课程地址**：在主界面的输入框中，粘贴您的小鹅通课程或专栏的网页链接。
2. **登录账号**：点击 **"登录"** 按钮，在弹出的窗口中点击「我的」，使用微信扫码完成登录。系统会自动保存 Cookie 与用户凭证。
3. **确认登录**：关闭登录窗口，主界面会显示为 "已登录" 状态。
4. **开始解析**：点击 **"开始解析"**。此时系统会打开内置解析窗口，**请耐心等待，切勿进行额外操作或关闭弹窗**。系统将自动完成：
   - 嗅探用户凭证 (`USER_ID`)
   - 自动滚动以加载所有分页目录
   - 逐一进入子页面抓取真实的 `m3u8` 地址
5. **导出列表**：所有视频地址解析完毕后，解析窗口会自动关闭，视频列表将自动填入下方的表格中。点击 **"导出"**（Export）按钮，系统会自动在你的**电脑「下载(Downloads)」目录**生成一份 `m3u8-list.txt` 文本文件。
6. **使用外部工具下载**：将导出的文本内容批量导入到专业的 M3U8 下载工具（推荐使用开源工具 [N_m3u8DL-RE](https://github.com/nilaoda/N_m3u8DL-RE)）中进行解密合并与下载。

### 🟡 场景二：重度定制化商铺（手动单集解析模式）

*手动抓包操作演示：*
[点击观看视频演示](https://github.com/user-attachments/assets/3c0af638-052e-4dce-9736-73a393697d71)

部分商家会深度定制店铺主页（链接常带有 `decorate/homepage` 等字样），具有多层嵌套和防刷机制。自动化爬虫可能无法识别其播放列表。对于这类课程，推荐**寻找其子级标准页面地址进行提取**，或使用以下**抓包方式逐一手动提取**：

1. **打开目标视频**：在电脑浏览器中，打开具体某一节视频播放页。
2. **唤出开发者工具**：按 `F12` 打开开发者工具，切换到 **网络（Network）** 面板。
3. **获取 m3u8 链接**：将筛选框改为 `m3u8`，点击播放视频，选中拦截到的 `.m3u8` 请求，其 **请求 URL** 即为真实的解析地址。
4. **提取完成**：复制获取到的地址，交给您的下载工具即可。

## 💡 其他补充说明

- **导出文件位置**：默认导出至操作系统的 `Downloads`（下载）目录，文件名为 `m3u8-list.txt`。
- **状态管理**：如需切换其他小鹅通账号，点击主界面登录状态旁的 "清除登录" 按钮即可。
